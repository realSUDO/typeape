# Changelog

## 1.0.0 — 2026-04-09
- Initial release
- 479k word dictionary, random stream (zero memory load)
- 3-row scrolling display with async prefetch
- WPM, raw WPM, accuracy, chars, consistency stats
- Timer: 15s / 30s / 45s / 60s / custom inline input box
- Blinking cursor on idle, steady on typing
- Terminal resize support with live re-wrap
- Quick restart with Enter, graceful exit with ESC
- Backspace to previous word only before first char of new word
